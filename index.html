Intended Features
Intended Features:
  • Simple sums possible.

  • Multiple calculations can be carried out without resetting page.

  • Ability to stack calculations, see answer without tapping '='.

  • Ability to do complex sums (currently bugging out when sums stacked)

  • Ability to do sums using floats or doubles (decimal point currently has no function attached)

  • Sums with multiple stages (e.g. 2+2x5) should be carried out in the correct order. Currently works left to right.

    • Will need to alter multiply and divide functionality to not show current sum until '=' is pressed. Otherwise shows calculation carried out out of order.
